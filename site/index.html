Interview Coach

Stay on point.

A desktop overlay that listens to interview questions in real time and surfaces the best example to lead with. One punchy sentence. No rambling.

View on GitHub Quick Start
Q

Listens for questions

Real-time speech-to-text via Deepgram Nova-3. Detects interview questions as they're asked on Zoom, Meet, or any call.

A

Surfaces your best answer

Matches questions against curated examples and local content folders. Shows the strongest lead sentence with source and metrics.

V

Knows who's talking

Speaker diarization identifies the interviewer vs. you, so it only reacts to their questions — not your own voice.

Quick start

1

Clone the repo and install:
git clone https://github.com/AllThePasswords/onpoint && cd onpoint && npm install

2

Get a free Deepgram API key at deepgram.com and add it to .env

3

Run npm start — the overlay appears in the bottom-right corner

4

Add your own content folders via Settings, customise the answer style, and you're on point.
